# Tutorial de Introducción


Descubramos **Docusaurus en menos de 5 minutos**.

## Primeros pasos

Empieza por **crear un nuevo sitio web**.

o **prueba Docusaurus ahora mismo** con **[docusaurus.new](https://docusaurus.new)**.

### Lo que necesitas

- [Node.js](https://nodejs.org/en/download/) version 20.0 o superior:
  - Cuando instalas Node.js, se recomienda marcar todas las casillas relacionadas con las dependencias.

## Genera un nuevo sitio

Genera un sitio usando el template **classic template**.

La plantilla clásica se añadirá automáticamente a tu proyecto una vez que ejecutes el comando:

```bash
npm init docusaurus@latest my-website classic
```

Puedes escribir este comando en el Símbolo del sistema, PowerShell, Terminal o cualquier otro terminal integrado en tu editor de código.

El comando también instala todas las dependencias necesarias para ejecutar Docusaurus.

## Comienza tu sitio web

Ejecuta el siguiente comando para iniciar la aplicación en el servidor:

```bash
cd my-website
npm run start
```

El comando `cd` cambia el directorio en el que estás trabajando. Para poder trabajar con tu sitio de Docusaurus recién creado, tendrás que desplazarte hasta allí en la terminal.

El comando `npm run start` compila tu sitio web localmente y lo aloja en un servidor de desarrollo, listo para que lo veas en http://localhost:3000/.

Abre `/intro.md` (this page) y edita algunas lineas: El sitio se  **recarga automaticamente** y muestra tus cambios.
